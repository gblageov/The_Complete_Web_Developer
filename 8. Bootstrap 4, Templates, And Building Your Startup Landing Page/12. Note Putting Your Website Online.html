If you are getting a 404 error when trying to put your website online to your Github pages, it means that Github cannot access your files. This means that there may be a few issues:

1. If your account on github has name: john, than your repository with your page name should be: john.github.io

2. You need to make sure you have an index.html  file spelled correctly inside of your project.

3. I show you an easier and nicer way to put your website online in the Git + Github section where you can add any repo you want to be hosted. However I recommend you wait until that section to get started with this.

If you have any questions, reach out to our student community in the #helpme or #github channel on Discord (Lecture 3 provides the link if you have not yet joined)!
